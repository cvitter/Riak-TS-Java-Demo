# Riak TS - Java Demonstration Code
Sample Java code that demonstrates how to use the Java client to work with Riak TS (Time Series) - Riak TS is a distributed NoSQL key/value store optimized for time series data. It provides a time series database solution that is extensible and scalable. For more information on Riak TS please the documentation homepage at: http://docs.basho.com/riakts/latest/
